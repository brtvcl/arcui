Button Component

The Arc UI provides a flexible and customizable button component for use in web applications.

The button component supports different visual styles, including primary, secondary, outline, and ghost.

Icon

Renders an icon within the button. icon can be an html string to render within the button. You can also make the button square with square property.

Icon Button

Square Button

Square button only works with icon

Props

Property Description Type
variant Variant of button "primary" | "secondary" | "outline" | "ghost"
text Inner text of button string
size Size of button "small" | "medium" | "large"
width CSS Width of button. "auto" for full width Css length as string e.g. "200px" | "5rem"
disabled Set button disabled boolean
loading Set button loading boolean
square Make button square boolean
icon Put icon inside before text. Can be html string string

Events

The .on() method attaches event handlers to the Arc UI components and returns an event listener id that you can use to remove the event listener with .off() method.
